Apa itu Navigation Bar?

Navigation bar (navbar) adalah sebuah elemen dalam antarmuka pengguna sebuah situs web atau aplikasi yang berfungsi sebagai menu navigasi untuk membantu pengguna berpindah antara halaman-halaman atau bagian-bagian yang berbeda dalam situs web atau aplikasi tersebut.

Struktur Navbar di HTML

Berikut hasilnya :

Penjelasan CSS untuk Navbar:
  • nav: Menetapkan latar belakang warna untuk navbar.
  • ul: Menghilangkan daftar item default dan padding.
  • li: Mengatur tampilan item menu secara horizontal.
  • a: Menetapkan tata letak tautan, padding, warna teks, dan menghilangkan dekorasi tautan.
  • a: hover: Mengubah warna latar belakang tautan saat dihover.
Dengan struktur HTML dan CSS di atas, Anda dapat membuat navbar yang sederhana dan mudah diubah sesuai kebutuhan desain Anda.
