Mengatur Ukuan dengan CSS Height-Width

Apa itu Height-Width?

Dalam CSS (Cascading Style Sheets), height dan width adalah properti yang digunakan untuk mengatur dimensi elemen HTML.

  • Height mengontrol tinggi elemen.
  • Width mengontrol lebar elemen.

Fungsi dan Kegunaan

  1. Menyesuaikan Ukuran Elemen: Mengatur height dan width memungkinkan Anda untuk menyesuaikan ukuran elemen HTML, seperti gambar, div, atau elemen lainnya.
  2. Responsif: Dengan menggunakan nilai relatif seperti persen (%) atau viewport units (vw dan vh), Anda dapat membuat desain web responsif yang akan menyesuaikan ukuran elemen dengan layar perangkat pengguna.
  3. Memastikan Konsistensi: Menggunakan height dan width membantu memastikan konsistensi tata letak dan tampilan elemen di seluruh situs web Anda.

Property yang digunakan.

  • Height: Properti height digunakan untuk menentukan tinggi suatu elemen. Ini dapat diatur dalam unit absolut seperti piksel (px), sentimeter (cm), atau dalam unit relatif seperti persen (%), viewport height (vh), dan viewport width (vw). Contohnya, height: 100px; akan membuat elemen memiliki tinggi 100 piksel.
  • Width: Properti width digunakan untuk menentukan lebar suatu elemen. Seperti halnya dengan height, width juga dapat diatur dalam unit absolut atau relatif. Misalnya, width: 50%; akan membuat elemen menempati 50% lebar dari parent container-nya.
Ini adalah dua properti dasar yang secara langsung mengontrol dimensi suatu elemen dalam hal tinggi dan lebar. Dengan menggunakan kedua properti ini, Anda dapat mengatur dan mengontrol tata letak elemen dengan tepat sesuai kebutuhan desain Anda.
