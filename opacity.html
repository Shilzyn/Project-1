Apa itu properti Opacity?

Opacity dalam CSS mengacu pada tingkat transparansi dari sebuah elemen HTML. Ini memungkinkan Anda untuk mengatur seberapa transparan atau tidak transparan suatu elemen, sehingga memengaruhi seberapa banyak konten di bawahnya terlihat melalui elemen tersebut.

Nilai yang dapat digunakan dalam properti CSS opacity adalah antara 0 hingga 1, di mana:

  • Nilai 0 menandakan elemen tersebut sepenuhnya transparan, sehingga tidak terlihat sama sekali.
  • Nilai 1 menandakan elemen tersebut sepenuhnya tidak transparan, sehingga terlihat dengan jelas tanpa adanya transparansi.
Selain nilai langsung 0 dan 1, nilai desimal antara 0 dan 1 juga dapat digunakan untuk mencapai tingkat transparansi yang diinginkan. Misalnya, 0.5 akan membuat elemen menjadi setengah transparan, sedangkan 0.75 akan membuatnya hampir tidak transparan, tetapi masih memiliki sedikit transparansi.

Contoh penggunaan nilai dalam properti opacity:

  • opacity: 0; - Membuat elemen sepenuhnya transparan.
  • opacity: 0.5; - Membuat elemen setengah transparan.
  • opacity: 1; - Membuat elemen sepenuhnya tidak transparan.
Properti opacity sangat berguna dalam desain web untuk menciptakan efek lapisan transparan, overlay, atau untuk menyoroti elemen tertentu dengan memperhatikan konten di bawahnya.
