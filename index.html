HTML

Pengenalan HTML: Bahasa Markup untuk Membangun Halaman Web

HTML, singkatan dari HyperText Markup Language, adalah bahasa markup yang digunakan untuk membuat dan memformat halaman web. Sebagai tulang punggung dari semua situs web di internet, pemahaman yang kuat tentang HTML adalah kunci bagi siapa pun yang ingin terlibat dalam pembuatan konten web. Artikel ini akan memberikan pemahaman rinci tentang HTML, mulai dari sejarahnya hingga prinsip-prinsip dasar dan penggunaannya dalam membangun halaman web.

Sejarah HTML

HTML pertama kali diperkenalkan oleh Tim Berners-Lee pada tahun 1991 sebagai bagian dari proyek World Wide Web di CERN (Organisasi Eropa untuk Riset Nuklir). Versi pertama HTML, HTML 1.0, hanya berisi sejumlah kecil tag dan memiliki kemampuan terbatas. Seiring berjalannya waktu, standar HTML terus berkembang melalui serangkaian versi, yang dipimpin oleh World Wide Web Consortium (W3C), hingga mencapai versi terbaru pada saat ini, HTML

Konsep Dasar HTML

HTML bekerja dengan cara menyusun elemen-elemen atau tag-tag untuk membentuk struktur dasar dari sebuah halaman web. Setiap tag memiliki peran dan fungsi tertentu dalam menentukan tampilan dan perilaku elemen yang terkait.

Struktur Dasar Dokumen HTML

Sebuah dokumen HTML umumnya terdiri dari elemen-elemen berikut:

- !DOCTYPE html: Mendefinisikan jenis dokumen HTML.
- html: Mengapit seluruh isi halaman web.
- head: Berisi informasi-informasi meta dan judul halaman.
- meta: Menyediakan metadata, seperti pengkodean karakter dan viewport.
- title: Menetapkan judul halaman.
- body: Mengapit konten yang akan ditampilkan di halaman web.
- header, nav, main, footer: Bagian-bagian struktural dari halaman web.

Tag dan Elemen HTML

Tag HTML adalah komponen dasar yang digunakan untuk membangun struktur halaman web. Beberapa tag umum meliputi:
- h1, h2, ..., h6: Judul berjenjang.
- p: Paragraf teks.
- a: Hyperlink.
- img: Gambar.
- div: Kontainer blok.
- span: Kontainer inline.
- ul, ol, li: Daftar tak terurut (bullet) dan terurut (numbered).

Atribut HTML

Atribut memberikan informasi tambahan tentang elemen tertentu. Contohnya adalah src pada tag (img) untuk menentukan sumber gambar atau href pada tag (a) untuk menentukan URL tujuan hyperlink.

Semantik HTML

Penggunaan tag sesuai dengan makna atau semantiknya yang sebenarnya membantu mesin pencari dan pembaca layar untuk memahami struktur dan konten halaman web. Hal ini penting untuk aksesibilitas dan SEO (Search Engine Optimization).

CSS

Mendalami CSS: Mempercantik Halaman Web dengan Gaya

Cascading Style Sheets (CSS) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dan format halaman web. Dengan CSS, pengembang web dapat mengendalikan tata letak, warna, font, dan berbagai aspek visual lainnya dari elemen HTML. Artikel ini akan menjelaskan secara rinci tentang CSS, termasuk sejarah, konsep dasar, serta hubungannya dengan HTML dalam membangun halaman web yang menarik dan berfungsi.

Sejarah CSS

CSS pertama kali diperkenalkan pada tahun 1996 oleh Hakon Wium Lie dan Bert Bos. Tujuan utama CSS adalah untuk memisahkan struktur konten dari presentasi visual, sehingga memungkinkan pengembang web untuk lebih mudah mengelola tampilan halaman web secara terpusat dan konsisten.

Konsep Dasar CSS

CSS bekerja dengan cara menetapkan aturan gaya (style rules) untuk menyesuaikan tampilan elemen HTML. Setiap aturan gaya terdiri dari sebuah selector yang menentukan elemen atau kelompok elemen yang akan diubah, serta properti-properti yang menetapkan nilai-nilai gaya.

Hubungan antara CSS dan HTML

HTML dan CSS bekerja bersama-sama untuk membuat halaman web yang menarik dan berfungsi. HTML digunakan untuk menentukan struktur konten, sedangkan CSS digunakan untuk mengatur tampilan dan gaya elemen-elemen tersebut.

Cara Menyisipkan CSS ke dalam HTML

Ada beberapa cara untuk menyisipkan CSS ke dalam dokumen HTML:
1. Inline CSS: Menyertakan gaya langsung di dalam atribut style pada elemen HTML.
2. Internal CSS: Menyertakan aturan gaya di dalam tag (style) di dalam elemen (head) dokumen HTML.
3. External CSS: Menyertakan file CSS terpisah yang dihubungkan dengan menggunakan tag (link) di dalam elemen (head) dokumen HTML.

Selektor CSS

Selektor CSS digunakan untuk menargetkan elemen-elemen tertentu di dalam dokumen HTML. Beberapa jenis selektor yang umum digunakan meliputi:
- Selector Elemen: Menargetkan elemen berdasarkan nama tagnya, misalnya p, h1, div, dll.
- Selector Kelas: Menargetkan elemen berdasarkan class yang diberikan, misalnya .class, dimana .class adalah nama class.
- Selector ID: Menargetkan elemen berdasarkan ID yang diberikan, misalnya #id, dimana #id adalah nama ID.
